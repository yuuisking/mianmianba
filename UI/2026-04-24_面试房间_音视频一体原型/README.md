# 面试房间 UI 原型（语音 + 视频一体）

## 交互说明（原型）
- **摄像头开关**：打开后显示本地摄像头画面；未打开则显示头像占位
- **麦克风开关**：打开后显示语音波动（采集到声音会有动态波形）
- **字幕区**：展示 AI 面试官提问的实时滚动字幕（原型用模拟数据）
- **计时器**：显示面试已进行时间

## 文件
- `interview-room.html`：入口页面（静态原型）
- `assets/anthropic-ui.css`：品牌风格 + 该页面组件样式
- `assets/interview-room.js`：摄像头/麦克风切换、波形动画、字幕滚动、计时器（原型）
