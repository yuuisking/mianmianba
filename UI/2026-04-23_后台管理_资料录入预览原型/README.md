# 后台管理 UI 原型（轻量资料录入）

## 目标
后台管理只做“最小闭环”：
1) 上传 **GitHub 文档链接** 或 **本地文档**  
2) 生成“归纳后的知识点预览”（典型回答/扩展/追问）  
3) 人工确认入库（写入体系化知识库）

> 不做花哨的内容编辑器；编辑动作应发生在“人工审核/确认”阶段的最小字段修正。

## 文件
- `admin.html`：入口
- `assets/admin-lite.js`：创建预览、创建队列任务、确认入库（原型用 LocalStorage 保存）
- `assets/anthropic-ui.css`：品牌风格 CSS（同学习中心）
